# 2주차: 문자열과 입/출력 (Strings and Input/Output)

## 학습 목표
이번 주에는 **문자열 조작**과 **사용자 입/출력**을 배웁니다!

### 주요 학습 내용
1. **문자열 기초** - 연결, 반복, 슬라이싱
2. **문자열 메서드** - upper, lower, replace 등
3. **입/출력 함수** - input(), print() 심화
4. **실습 프로젝트** - 간단한 대화형 프로그램
